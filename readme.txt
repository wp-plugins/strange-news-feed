== Description ==

The Strange News Feed plugin will pull strange news stories and odd products from gloomwire.com. The content can be displayed in a widget or on a post/page using short code.


== Installation ==

1. Upload 'strange-news-feed' directory to your `/wp-content/plugins` folder
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode in your posts or pages: [strange-news-feed=limit="10" show_thumbnail="0" hide_description="0" show_date="0"]
4. Use the widget from the widget menu in the Wordpress admin pages
